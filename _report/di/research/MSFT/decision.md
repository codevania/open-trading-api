## Rationale

1. MSFT remains a high-quality satellite candidate because Microsoft 365, Azure, security, GitHub, Dynamics, and LinkedIn create a broad enterprise software and cloud platform rather than a single-product AI bet.
2. The current located price of `$384.36` is not an obvious bargain. The valuation note records FY2025 P/E near `28.0x`, EV/FCF near `39.3x`, and a low FCF yield while AI capex is elevated.
3. The research-target ETF overlap is already `4.39pp`, so any single-name MSFT exposure would add to existing core U.S. equity and Nasdaq-style exposure.
4. For this single-name route, the account path is taxable overseas-stock brokerage exposure. ISA, pension, and IRP remain better suited to separate Korea-listed ETF/fund wrapper comparisons.
5. This note creates a monitored candidate, not an order. Actual execution requires a same-day quote, portfolio exposure refresh, broker/tax confirmation, and a new decision entry.
