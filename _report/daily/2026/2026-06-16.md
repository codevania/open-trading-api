## 9. 데이터 품질 메모

- 입력 파일: 루트 [[_report/watchlist.yaml|_report/watchlist.yaml]]과 `_report/routines/daily-report-routine.md`가 없어 [[_report/di/watchlist.yaml|_report/di/watchlist.yaml]]과 [[_report/di/routines/daily-report-routine|_report/di/routines/daily-report-routine.md]]를 사용했다. 산출물은 사용자 요청대로 루트 [[_report/daily/2026/2026-06-16|_report/daily/2026/2026-06-16.md]]와 [[_report/decisions/decision-log|_report/decisions/decision-log.md]]에 저장한다.
- KIS preflight: 국내 `inquire_price`, `inquire_daily_itemchartprice`, `investor_trade_by_stock_daily`, `news_title`, 해외 `price`, `dailyprice`는 호출 전 로컬 MCP `find_api_detail` 핸들러로 상세 정보를 저장했다.
- 네트워크: 최초 sandbox 실행은 Windows socket 권한으로 실패했고, 승인된 네트워크 실행에서 같은 조회성 API가 정상 수집됐다.
- NVDA: 2026-06-16 최신 행은 거래량 0이라 미국장 확정 데이터로 보지 않는다. 거래량이 있는 최신 행은 2026-06-15다.
- 뉴스/공시: 이번 요청의 필수 범위가 현재가/기간별 시세/수급이어서 실제 뉴스 API는 호출하지 않았다.
- 실제 주문 API: 호출하지 않았다.
