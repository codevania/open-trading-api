## 9. 데이터 품질 메모

- 입력 파일: 루트 [[_report/watchlist.yaml|_report/watchlist.yaml]]과 `_report/routines/daily-report-routine.md`가 없어 [[_report/di/watchlist.yaml|_report/di/watchlist.yaml]]과 [[_report/di/routines/daily-report-routine|_report/di/routines/daily-report-routine.md]]를 사용했다. 산출물은 사용자 요청대로 루트 [[_report/daily/2026/2026-06-17|_report/daily/2026/2026-06-17.md]]와 [[_report/decisions/decision-log|_report/decisions/decision-log.md]]에 저장한다.
- KIS preflight: 현재 Codex App 표면에는 직접 KIS MCP 도구가 노출되지 않아, repo-local MCP `configs/*.json`의 `find_api_detail` 동등 정보를 저장한 뒤 checked-in `examples_llm` 조회성 래퍼만 호출했다.
- NVDA: 2026-06-17 최신 행은 거래량 0이라 미국장 확정 데이터로 보지 않는다. 거래량이 있는 최신 행은 2026-06-16이다.
- 해외 `dailyprice`: pagination 결과에 중복 날짜가 있어 차트와 판단에는 날짜 기준으로 중복 제거한 값만 사용했다.
- 뉴스/공시: 이번 요청의 필수 범위가 현재가/기간별 시세/수급이어서 실제 뉴스 API는 호출하지 않았다.
- 실제 주문 API: 호출하지 않았다.
